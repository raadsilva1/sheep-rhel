## Tags (Workspaces)

River uses **tags** instead of workspaces. A window can belong to one or more tags, and one or more tags can be visible at once.

| Shortcut | Action |
|----------|--------|
| `Super + 1` … `Super + 9` | Focus tag 1–9 |
| `Super + 0` | Focus **all** tags |
| `Super + Shift + 1` … `Super + Shift + 9` | Move focused window to tag 1–9 |
| `Super + Ctrl + 1` … `Super + Ctrl + 9` | Toggle tag visibility (show/hide) |
| `Super + Shift + Ctrl + 1` … `Super + Shift + Ctrl + 9` | Toggle window membership on tag |

> **Tag math:** Tag *N* = bit `1 << (N-1)`. Tag 1 = `0x1`, Tag 2 = `0x2`, Tag 3 = `0x4`, etc.

## Launcher

`wmenu-run` is a Wayland-native `dmenu` replacement. It is themed automatically to match your selected River theme (background, foreground, selection, and prompt colors). It reads executables from `$PATH` and presents them in a searchable menu at the top of the screen.

## Mouse

River Classic supports basic mouse actions out of the box:

- **Left-click** a window → focus it
- **Super + Left-drag** → move floating window
- **Super + Right-drag** → resize floating window

> Tiled windows are managed by the layout generator; floating windows can be moved/resized freely.

## Concepts

### Tags vs. Workspaces
Traditional WMs have workspaces: a window lives on *one* workspace.  
River has **tags**: a window can be on *multiple* tags, and you can view *multiple* tags at once. Think of tags like filters.

### Main / Stack Layout
`rivertile` divides the screen into:
- **Main area** (left, configurable width) — holds your primary windows
- **Stack area** (right) — holds remaining windows stacked vertically

### Configuration
Your init script lives at:
```
~/.config/river/init
```
Edit it and re-log to apply changes, or run:
```bash
~/.config/river/init
```
from an existing River session to reload keybindings live.

## Themes

Switch themes anytime with:
```bash
sheep-rhel --theme <name>
```
Or interactively:
```bash
sheep-rhel --interactive
```

Available themes: `rhel-red` (default), `charcoal`, `fedora-blue`, `light-gray`, `accent-orange`, `dark-red`, `steel`.
